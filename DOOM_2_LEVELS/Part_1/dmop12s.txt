This is an MSDOS based DOOM and HERETIC utility.

This program can place new objects into new locations in any DOOM or 
Heretic IWAD or Patch wad level.

Shareware version 1.12 of the Object Placer for DOOM 1, DOOM 2, 
ALIENTC, and Heretic by Dusty Bedford

There is an example of a large wad which has been altered in the zip 
file.
